# HR Analytics Dashboard

## Overview
This project is an interactive dashboard created to explore and present employee attrition insights in a more visual and intuitive way.

Instead of working only with raw data and notebooks, the goal was to take key observations and represent them in a clean browser-based interface that highlights important patterns in employee behavior.

## What this dashboard shows
The dashboard focuses on a few key questions:

- How many employees are leaving compared to those staying  
- Whether age shows any visible influence on attrition  
- If overtime is associated with higher attrition  

These are presented through simple visualizations so that the patterns are easy to understand at a glance.

## Features
- Attrition distribution overview  
- Age group analysis  
- Overtime and attrition comparison  
- Clean and responsive layout  
- Structured and readable interface  

## Tech Stack
- HTML for structure  
- CSS for layout and styling  
- JavaScript for logic and rendering  
- Chart.js for visualizations  

## How to run
1. Clone the repository  
2. Open the project folder  
3. Open `index.html` in your browser  

No additional setup is required.

## Key Observations
Based on the current visualizations:

- Employees working overtime appear more likely to leave  
- Certain age groups show higher attrition patterns  
- Attrition is influenced by multiple factors rather than a single variable  

## Why this project
The idea behind this project was to move beyond analysis and focus on presentation.

Instead of keeping insights limited to notebooks, this dashboard makes them easier to interpret and more accessible through a simple interface.
